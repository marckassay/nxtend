Create a page in an existing Ionic Angular app.

## Usage

```
nx generate @nxtend/ionic-angular:page --name MyPage --project myApp
```

## Options

### --directory

Alias(es): d

Default: `null`

Type: `string`

The subdirectory of the new page relative to the `app` folder
